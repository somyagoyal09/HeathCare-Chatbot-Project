# 🩺 AI Healthcare Chatbot

Ever wondered if Artificial Intelligence can assist in basic healthcare guidance? 🤖  
This project presents an **AI-powered Healthcare Chatbot** that predicts possible diseases based on symptoms provided by the user.

The system uses **Machine Learning (Random Forest Classifier)** trained on a dataset of symptoms and diseases.  
It interacts with users through a **chatbot interface**, asks follow-up questions, and provides predicted diseases along with precautionary suggestions.

---

## 🚀 What this project demonstrates

✅ Symptom extraction from user input  
✅ Machine Learning model for disease prediction  
✅ Interactive chatbot that asks follow-up questions  
✅ Confidence-based disease prediction  
✅ Health tips and precautions for predicted diseases  
✅ Web integration using Flask for real-time interaction

---

## 🧠 Learning Outcomes

Through this project, the following concepts are explored:

- Understanding and preprocessing healthcare datasets
- Training machine learning models for symptom-based disease prediction
- Building an interactive chatbot interface
- Integrating ML models with a web application
- Creating an AI system capable of real-time user interaction
